# 决策树：docflow-investigate

## 调查路径判断

[用户请求属于“调查”]
├── 否：返回“不触发”，建议改走执行类技能。
└── 是：继续
    └── [llmdoc 是否可用]
        ├── 否：声明证据不足并进入代码检索。
        └── 是：先阅读 index + overview + 相关文档。
            └── [文档是否已能回答问题]
                ├── 是：直接输出结论与文档证据。
                └── 否：进入代码检索并补充证据。
                    └── [是否需要外部信息]
                        ├── 否：整理结论并输出。
                        └── 是：声明边界后再进行外部检索。
