# 公众号选题策划框架

## 选题四象限

```
        高时效性
           │
    热点追踪│观点输出
           │
低深度 ────┼──── 高深度
           │
    娱乐轻松│干货教程
           │
        低时效性
```

| 象限 | 特点 | 适合场景 | 生命周期 |
|------|------|----------|----------|
| 热点追踪 | 时效高+深度低 | 新闻评论、热点解读 | 1-3 天 |
| 观点输出 | 时效高+深度高 | 趋势分析、深度评论 | 1-2 周 |
| 娱乐轻松 | 时效低+深度低 | 段子、趣闻、盘点 | 长期有效 |
| 干货教程 | 时效低+深度高 | 方法论、教程、指南 | 长期有效 |

## 选题五维度

### 1. 热点追踪

**来源**：
- 微信热搜、微博热搜
- 行业新闻、产品发布
- 政策变化、市场动态

**切入方式**：
- 第一时间解读
- 独特角度评论
- 关联已有内容

### 2. 痛点解决

**挖掘方法**：
- 评论区高频问题
- 社群讨论热点
- 搜索关键词分析
- 竞品内容空白

**问题模板**：
```
"如何 [目标]？"
"为什么 [现象]？"
"[问题] 怎么解决？"
```

### 3. 认知颠覆

**类型**：
- 打破常见误区
- 反直觉的真相
- 行业内幕揭秘

**句式**：
```
"你以为的 X，其实是 Y"
"99% 的人都错了的 X"
"X 的真相，可能和你想的不一样"
```

### 4. 情感共鸣

**类型**：
- 职场焦虑
- 成长困惑
- 身份认同
- 人生选择

**触发点**：
- 特定人群的普遍经历
- 时代背景下的共同情绪
- 人生阶段的典型困惑

### 5. 实用价值

**类型**：
- 工具推荐
- 方法论
- 清单指南
- 模板资源

**标题模板**：
```
"X 个 [工具/方法]，帮你 [目标]"
"[目标] 的完整指南"
"[场景] 必备的 X 个 [资源]"
```

## 选题评估矩阵

对每个选题进行 1-5 分评估：

| 维度 | 评估问题 | 权重 |
|------|----------|------|
| 相关性 | 与账号定位匹配度？ | 25% |
| 需求度 | 目标读者有多需要？ | 25% |
| 独特性 | 与竞品差异化程度？ | 20% |
| 可操作性 | 我能写好这个选题吗？ | 15% |
| 传播性 | 读者愿意分享吗？ | 15% |

**总分计算**：
```
总分 = 相关性×0.25 + 需求度×0.25 + 独特性×0.20 + 可操作性×0.15 + 传播性×0.15
```

**决策标准**：
- ≥4.0：优先写作
- 3.0-3.9：备选池
- <3.0：暂不考虑

## 选题风暴流程

### Step 1: 发散（10分钟）

使用以下触发器生成 10-15 个选题：

```
1. 最近遇到的问题：___
2. 最近学到的东西：___
3. 最近看到的热点：___
4. 读者常问的问题：___
5. 竞品没写好的主题：___
6. 我的独特经历/观点：___
```

### Step 2: 筛选（5分钟）

用评估矩阵快速打分，保留前 5 个。

### Step 3: 深化（10分钟）

对 Top 5 选题进行角度细化：

```
选题：___
可能的角度：
1. [人群角度] 针对谁？
2. [问题角度] 解决什么？
3. [方法角度] 用什么方式？
4. [结果角度] 能得到什么？
```

### Step 4: 确定（5分钟）

选择最终选题 + 确定切入角度 + 拟定初步标题。

## 常见选题陷阱

| 陷阱 | 表现 | 解决方法 |
|------|------|----------|
| 太宽泛 | "如何学好编程" | 细化：针对谁、什么场景 |
| 太小众 | 只有极少数人关心 | 提升抽象层级，扩大受众 |
| 无差异 | 竞品已写很多 | 找独特角度或放弃 |
| 无价值 | 写了也没人看 | 回归读者痛点 |
| 无能力 | 超出能力范围 | 选择擅长领域 |
