# 会议模板集

## 会议类型模板

### 1. 项目启动会议

```markdown
# 项目启动会议纪要

**项目名称**: [项目名]
**日期**: YYYY-MM-DD
**参与者**: [姓名列表]
**主持人**: [姓名]

## 项目概述

### 项目背景

[项目的背景信息和驱动因素]

### 项目目标

- 目标 1
- 目标 2
- 目标 3

### 项目范围

**包含**:

- 范围项 1
- 范围项 2

**不包含**:

- 排除项 1
- 排除项 2

## 关键里程碑

| 里程碑   | 预计日期 | 交付物   |
| -------- | -------- | -------- |
| 阶段 1   | MM-DD    | 交付物 1 |
| 阶段 2   | MM-DD    | 交付物 2 |
| 项目上线 | MM-DD    | 最终交付 |

## 团队分工

| 角色       | 成员   | 职责     |
| ---------- | ------ | -------- |
| 项目经理   | [姓名] | 整体协调 |
| 技术负责人 | [姓名] | 技术决策 |
| 开发成员   | [姓名] | 功能开发 |

## 风险识别

| 风险   | 概率     | 影响     | 应对措施 |
| ------ | -------- | -------- | -------- |
| 风险 1 | 高/中/低 | 高/中/低 | 措施 1   |

## 下一步行动

| 任务   | 负责人 | 截止日期 |
| ------ | ------ | -------- |
| 任务 1 | [姓名] | MM-DD    |
| 任务 2 | [姓名] | MM-DD    |

## 下次会议

- **时间**: YYYY-MM-DD HH:MM
- **议题**: [主要议题]
```

### 3. 冲刺回顾会议

```markdown
# Sprint [N] 回顾会议

**日期**: YYYY-MM-DD
**Sprint 周期**: MM-DD 至 MM-DD
**参与者**: [团队成员]
**主持人**: [姓名]

## Sprint 数据

| 指标   | 计划 | 完成 | 达成率 |
| ------ | ---- | ---- | ------ |
| 故事点 | X    | Y    | Z%     |
| 任务数 | X    | Y    | Z%     |

## 回顾讨论

### 😊 做得好的 (Continue)

- 好的方面 1
- 好的方面 2

### 😐 需要改进的 (Stop)

- 问题 1
- 问题 2

### 💡 新尝试 (Start)

- 改进措施 1
- 改进措施 2

## 改进行动

| 改进项 | 责任人 | 验证方式   | 截止日期 |
| ------ | ------ | ---------- | -------- |
| 改进 1 | [姓名] | [如何验证] | MM-DD    |
| 改进 2 | [姓名] | [如何验证] | MM-DD    |

## 团队心情指数
```

😃 非常好: ████ 2人
😊 良好: ████████ 4人
😐 一般: ████ 2人
😟 不佳: 0人

```

## 下次回顾

- **时间**: YYYY-MM-DD
- **关注点**: [上次改进项的效果]
```

### 4. 技术评审会议

```markdown
# 技术评审会议纪要

**日期**: YYYY-MM-DD
**评审主题**: [主题名称]
**参与者**: [姓名列表]
**记录人**: [姓名]

## 评审目标

[本次评审的目的]

## 技术方案概述

### 背景

[问题背景]

### 方案描述

[技术方案简述]

### 架构图

[架构图或链接]

## 评审讨论

### 讨论点 1: [主题]

**问题**: [提出的问题]
**讨论**: [讨论过程]
**结论**: [达成的结论]

### 讨论点 2: [主题]

...

## 评审结论

| 方面     | 评估         | 备注 |
| -------- | ------------ | ---- |
| 可行性   | ✅ / ⚠️ / ❌ | 说明 |
| 性能     | ✅ / ⚠️ / ❌ | 说明 |
| 安全性   | ✅ / ⚠️ / ❌ | 说明 |
| 可维护性 | ✅ / ⚠️ / ❌ | 说明 |

**最终决定**:

- [ ] 通过
- [ ] 有条件通过 (需要修改后复评)
- [ ] 不通过 (重新设计)

## 待办事项

| 任务   | 负责人 | 截止日期 |
| ------ | ------ | -------- |
| 任务 1 | [姓名] | MM-DD    |

## 复评安排

- **时间**: YYYY-MM-DD (如需要)
- **复评范围**: [修改内容]
```

## 议程模板

### 通用议程

```markdown
# 会议议程

**会议主题**: [主题]
**日期时间**: YYYY-MM-DD HH:MM
**地点/链接**: [地点/会议链接]
**主持人**: [姓名]

## 参与者

| 姓名   | 角色   | 必需/可选 |
| ------ | ------ | --------- |
| [姓名] | [角色] | 必需      |
| [姓名] | [角色] | 可选      |

## 会前准备

- [ ] 阅读材料: [链接]
- [ ] 准备数据: [具体要求]

## 议程

| 时间          | 议题   | 负责人 | 目标       |
| ------------- | ------ | ------ | ---------- |
| HH:MM (5min)  | 开场   | [姓名] | 对齐目标   |
| HH:MM (15min) | 议题 1 | [姓名] | [预期产出] |
| HH:MM (20min) | 议题 2 | [姓名] | [预期产出] |
| HH:MM (15min) | 讨论   | 全体   | 达成共识   |
| HH:MM (5min)  | 总结   | [姓名] | 明确行动项 |

## 预期产出

- 产出 1
- 产出 2

## 备注

[其他需要说明的事项]
```
