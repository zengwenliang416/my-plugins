# 微信公众号写作风格指南

## 平台特性

- **用户心态**: 碎片化阅读，信息密度要求高
- **阅读场景**: 通勤、午休、睡前
- **内容偏好**: 干货型、情感共鸣型、观点输出型

## 标题规范

### 必备元素（选 1-2）

- 数字：「3 个方法」「90% 的人不知道」
- 悬念：「真相是...」「结果让人意外」
- 痛点：「别再...了」「为什么总是...」
- 利益：「省下 XXX」「提升 XX%」

### 字数

- 推荐：15-25 字
- 最长：不超过 32 字

## 正文结构

### 开篇（前 3 句定生死）

```
Hook → 痛点/情境 → 承诺价值
```

### 主体

- 段落短小：3-5 句一段
- 小标题醒目：引导快速浏览
- 金句点缀：每 500 字一个
- 案例/数据：增加可信度

### 收尾

- 总结核心观点
- 明确 CTA（关注/转发/评论）

## 语言风格

| 要素 | 规范                   |
| ---- | ---------------------- |
| 人称 | 第二人称「你」拉近距离 |
| 句式 | 短句为主，长句拆分     |
| 语气 | 亲切但专业，避免说教   |
| 禁忌 | 敏感词、绝对化表述     |

## 字数参考

| 类型     | 字数      |
| -------- | --------- |
| 干货长文 | 2500-4000 |
| 观点输出 | 1500-2500 |
| 情感共鸣 | 1000-2000 |

## 配图要求

- 封面图：16:9，高清，有冲击力
- 正文插图：辅助理解，不喧宾夺主
- 图片数量：3-8 张
