# 小红书写作风格指南

## 平台特性

- **用户心态**: 种草、发现、分享
- **阅读场景**: 休闲浏览，视觉驱动
- **内容偏好**: 实用攻略、真实体验、颜值即正义

## 标题规范

### 必备元素

- Emoji：开头或关键词旁
- 关键词：直击搜索意图
- 数字：具体化价值

### 模板

```
[emoji] [核心关键词] | [价值点/数字]
```

### 示例

- 🔥 程序员转型指南 | 3个月涨薪50%
- ✨ 高效学习法 | 亲测有效！
- 💡 Claude Code 使用技巧 | 效率翻倍

## 正文结构

### 开篇（直接切入）

- 不要铺垫，第一句就给价值
- 可用疑问句引发共鸣

### 主体

- **短句为王**：一句一行
- **分点列出**：1️⃣ 2️⃣ 3️⃣
- **emoji 点缀**：每段 1-2 个
- **口语化**：像聊天一样

### 收尾

- 总结要点
- 互动引导：「你们觉得呢？」「收藏备用」

## 语言风格

| 要素  | 规范                        |
| ----- | --------------------------- |
| 人称  | 「我」「姐妹们」「宝子们」  |
| 句式  | 极短，一句一行              |
| 语气  | 热情、真诚、像朋友聊天      |
| emoji | 必备，但不过度（每段1-3个） |

## 字数参考

| 类型     | 字数     |
| -------- | -------- |
| 种草笔记 | 300-600  |
| 攻略教程 | 500-1000 |
| 深度分享 | 800-1500 |

## 配图要求

- **封面图**：3:4 或 1:1，高颜值，有信息量
- **图片数量**：3-9 张
- **图片风格**：统一滤镜，干净清爽
- **文字图**：关键信息可做成图片

## 话题标签

- 核心标签：2-3 个
- 长尾标签：3-5 个
- 热门标签：1-2 个（蹭流量）
