# 质量检查清单

交付前必须通过的检查项。

---

## 视觉质量

### 设计一致性

- [ ] 选定的风格方向贯彻到底
- [ ] 配色方案统一，无随意颜色
- [ ] 字体选择符合推荐，避开禁用字体
- [ ] 间距遵循 4px/8px 系统

### 避免 AI 通用美学

- [ ] 未使用 Inter/Roboto/Arial
- [ ] 未使用紫色渐变 + 白底
- [ ] 未使用居中卡片 + 大圆角 + 浅阴影
- [ ] 未使用对称三列网格

### 视觉层次

- [ ] 主次分明，不是所有元素同等重要
- [ ] 关键 CTA 突出
- [ ] 留白合理

---

## 交互质量

### 反馈状态

- [ ] 所有可点击元素有 hover 状态
- [ ] hover 状态有意义（非统一 scale-105）
- [ ] cursor 正确（pointer/default/not-allowed）
- [ ] focus 状态明显可见

### 动效

- [ ] 动效时长差异化（150ms/250ms/400ms）
- [ ] 使用差异化缓动函数
- [ ] 页面加载有错开动画
- [ ] 无不必要的动效

### 加载状态

- [ ] 按钮有 loading 状态
- [ ] 列表有 skeleton/placeholder
- [ ] 图片有 loading 占位

---

## 技术质量

### 代码规范

- [ ] 使用 SVG 图标，禁止 emoji
- [ ] CSS 变量定义颜色
- [ ] 类名语义化

### 响应式

- [ ] 手机端测试通过
- [ ] 平板端测试通过
- [ ] 桌面端测试通过

### 性能

- [ ] 图片优化（WebP/AVIF）
- [ ] 字体预加载
- [ ] 无布局抖动（CLS）

---

## 无障碍

### 对比度

- [ ] 正文对比度 ≥ 4.5:1
- [ ] 大标题对比度 ≥ 3:1
- [ ] 交互元素对比度 ≥ 3:1

### 键盘导航

- [ ] Tab 顺序合理
- [ ] 焦点指示器明显
- [ ] 可通过 Enter/Space 激活

### 语义化

- [ ] 正确使用 heading 层级
- [ ] 图片有 alt 文本
- [ ] 表单有 label
- [ ] ARIA 标注交互元素

---

## 暗色模式

### 颜色适配

- [ ] 背景非纯黑（使用 #0a0a0a 等）
- [ ] 文字非纯白（使用 #fafafa 等）
- [ ] 阴影调整透明度
- [ ] 边框使用半透明白

### 切换

- [ ] 平滑过渡
- [ ] 图标/图片适配

---

## 细节打磨

### 间距

- [ ] 所有间距符合 4px 基础单位
- [ ] 相关元素间距一致
- [ ] 组件内 padding 均匀

### 对齐

- [ ] 文本基线对齐
- [ ] 图标与文字垂直居中
- [ ] 网格对齐

### 边界情况

- [ ] 长文本溢出处理
- [ ] 空状态设计
- [ ] 错误状态设计
- [ ] 极端数据测试

---

```bash
npx lighthouse http://localhost:3000 --view
```

- WebAIM Contrast Checker
- Chrome DevTools > Rendering > CSS Overview

### 响应式测试

- Chrome DevTools 设备模式
- 真机测试
